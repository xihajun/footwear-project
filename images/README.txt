Machine Learning Applicants Task
================================

This zip file contains images of six shoeprints repeated six times each.

The filenames are in the following format:
submissionID-EUshoeSize-MAKE-makeName-ageInMonths-activity-repeatNo.jpeg

Where for this filename:
sid00000069-EU35.5-MAKE-Primark-M012-PRINT_STANDING-REP1.jpeg

The extacted fields are:

submissionID = sid00000069
EUshoeSize = 35.5
makeName = Primark
ageInMonths = 12
activity = PRINT_STANDING
repeatNo = 1

This information is not necessarily required for the task at hand, but
it allows the grouping of the shoe types and the order they were taken.

Enjoy the task!
